# Plan 04: Enhanced Error Messages with Suggestions

## Objective

Improve error messages for common CLI mistakes: file-not-found suggests similar files via Levenshtein distance, missing arguments hints `--help`. Keep consistent ANSI-colored formatting.

## Rationale

DX polish item. Better error messages reduce friction for new users. The "Did you mean..." pattern is standard in modern CLI tools (npm, pnpm, eslint).

## must_haves

- File not found errors include similar file suggestions when matches exist
- Missing required argument errors include `--help` hint
- Existing error color scheme preserved (red ✗, yellow ⚠, green ✓)

<read_first>

- src/util/ (existing utility files, if any)
</read_first>

<action>
Create `src/util/suggest.ts`:

1. Export `levenshtein(a: string, b: string): number`:
   - Standard Levenshtein distance algorithm
   - No external dependencies
   - Pure function, no side effects

2. Export `suggestSimilarFiles(inputFile: string, directory: string): string[]`:
   - If `directory` doesn't exist, return `[]`
   - Read directory entries with `fs.readdirSync()`
   - Filter to files with the same extension as inputFile (default `.html` if no extension)
   - Compute Levenshtein distance between `path.basename(inputFile)` and each filename
   - Return top 3 matches sorted by score (ascending)
   - Only return matches where score < basename.length * 0.6 (similarity threshold)
   - Return full relative paths (e.g., `./fixtures/simple.html`)
</action>

- `suggestSimilarFiles('nonexistent.html', '.')` returns `[]` or relevant suggestions
- When `./fixtures/simple.html` exists and user types `./fixtures/simple.htm`, suggestion includes `./fixtures/simple.html`
- TypeScript compilation passes
</acceptance_criteria>

<action>
Modify `src/cli/commands/convert.ts`:

1. Import `suggestSimilarFiles` from `../util/suggest.js`

2. Enhance file-not-found error (replace current simple check):
   ```typescript
   if (!fs.existsSync(file)) {
     spinner?.stop(); // Safe call if spinner hasn't started yet
     const dir = path.dirname(path.resolve(file));
     const suggestions = suggestSimilarFiles(file, dir);
     if (suggestions.length > 0) {
       showError(`File not found: ${file}\n  Did you mean: ${suggestions[0]}?`);
     } else {
       showError(`File not found: ${file}\n  Run 'h2ui --help' for usage information.`);
     }
     process.exit(1);
   }
   ```
